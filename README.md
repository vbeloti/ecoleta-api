# Ecoleta Api

## Open Endpoints

Open endpoints require no Authentication.

* [Items]: `GET /items` -- List all items
* [Points]: `GET /points` -- List all points
* [Points]: `GET /points/:id` -- List one point
* [Points]: `POST /points` -- Create one point
